# VibeChannel — план реализации

## Зафиксированные решения

- Репозиторий: публичный.
- Название: VibeChannel.
- Адрес приложения: https://brianfox.ru/vibechannel
- Ссылка с главной: https://brianfox.ru/
- Стек: React + TypeScript + SQLite.
- Подход: строгий TDD.
- Перед реализацией: согласовать HTML-мокап.
- Поддержка смартфонов: обязательная, Mobile First.

## MVP

1. Главный экран радиоплеера.
2. Режимы настроения: Ночь, Кодинг, Дождь, Киберпанк, Спокойно, Джаз.
3. Каталог станций из SomaFM и Radio Browser.
4. Избранное.
5. История прослушивания.
6. Заметка вечера.
7. SQLite для локальных серверных данных.

## TDD-правило

Никакого production-кода без сначала упавшего теста.
