## Variant: Cozy Radio

### Design stance
Ламповый вечерний радиоприёмник, больше тепла и меньше техничности.

### Key choices
- Layout: большой «приёмник» + полка станций.
- Typography: заголовки с засечками, интерфейсные элементы системным шрифтом.
- Color: дерево, янтарь, кремовый текст.
- Interaction: play/pause, режимы, избранное, подбор станции.

### Trade-offs
- Strong at: настроение, уют, личный характер.
- Weak at: может казаться менее современным и менее «кодерским».

### Best for
Если хочется, чтобы проект был не инструментом, а маленьким местом отдыха.
