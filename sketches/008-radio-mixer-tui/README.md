## Variant: Radio Mixer TUI

### Design stance
Терминальный радиомикшер: не карточки, а каналы, шкала частот и уровни.

### Key choices
- Layout: крупный тюнер + микшер каналов.
- Typography: моноширинная техническая сетка.
- Color: янтарный терминал.
- Interaction: выбор канала, play/pause, scan, memory.

### Trade-offs
- Strong at: радио-вайб самый явный.
- Weak at: чуть дальше от классического ncurses.
