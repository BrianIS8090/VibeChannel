## Variant: Night Console

### Design stance
Техничный ночной пульт, почти терминал для радиоэфира.

### Key choices
- Layout: консольный экран + панели режимов и очереди.
- Typography: моноширинный шрифт.
- Color: почти чёрный, зелёный, циан, розовый сигнал.
- Interaction: run/pause stream, scan, режимы, очередь.

### Trade-offs
- Strong at: уникальность, кодерский вайб, узнаваемость.
- Weak at: менее дружелюбно для обычного пользователя, чуть холодно.

### Best for
Если проект должен выглядеть как личный «ночной инструмент» Брайна.
