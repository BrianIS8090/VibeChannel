## Variant: Terminal TUI Split

### Design stance
Веб-версия TUI/ncurses: панели, горячие клавиши, список станций и лог сессии.

### Key choices
- Layout: три панели на десктопе, один потоковый экран на телефоне.
- Typography: полный моноширинный интерфейс.
- Color: почти монохромный зелёный терминал.
- Interaction: play, random, save, выбор станций и настроений, лог событий.

### Trade-offs
- Strong at: максимально терминальный и функциональный.
- Weak at: может казаться сухим и менее «музыкальным».

### Best for
Если хочется прям «радио внутри терминала», без лишней красоты.
