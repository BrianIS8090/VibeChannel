# VibeChannel — настоящий TUI-раунд

Этот раунд сделан после замечания: предыдущие варианты были терминальными только по эстетике, но структура оставалась обычным веб-приложением.

Здесь наоборот: структура ближе к реальному терминальному TUI, а радио-вайб добавлен через частоты, тюнер, VU-метры, очередь эфира и сессионный лог.

## Варианты

1. `../007-radio-tui-classic/index.html` — классический TUI: список станций / плеер / лог.
2. `../008-radio-mixer-tui/index.html` — TUI радиомикшер: тюнер, каналы, уровни.
3. `../009-pocket-tui-radio/index.html` — мобильный карманный TUI: проще, чище, меньше панелей.

## Моё мнение

Лучший кандидат — `007-radio-tui-classic`: он ближе всего к настоящему TUI и при этом сохраняет радио-ощущение. `008` интереснее как радио-микшер, `009` лучше для телефона, но чуть менее «настоящий терминал».
