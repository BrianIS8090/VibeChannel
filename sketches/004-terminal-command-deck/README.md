## Variant: Terminal Command Deck

### Design stance
Серьёзный терминальный пульт: выглядит как рабочая консоль управления эфиром, но остаётся понятным на телефоне.

### Key choices
- Layout: крупный терминальный экран + боковые панели очереди и режимов.
- Typography: моноширинный интерфейс, команды и логи.
- Color: чёрный/зелёный, лёгкий кислотный акцент.
- Interaction: run/pause, scan, save fav, выбор станции, выбор режима.

### Trade-offs
- Strong at: самый зрелый и «продуктовый» из терминальных вариантов.
- Weak at: менее игрушечный, может быть слишком строгим.

### Best for
Если VibeChannel должен ощущаться как личная утилита Брайна, а не декоративный плеер.
