## Variant: Pixel Pocket Radio

### Design stance
Минимальный карманный пиксельный радиоплеер: как маленькая игровая консоль с эфиром.

### Key choices
- Layout: один крупный handheld-экран, снизу 3 действия, сбоку короткий список станций.
- Typography: моноширинный пиксельный характер без перегруза.
- Color: ночной фиолетовый, жёлтый корпус, розовый акцент.
- Interaction: play, scan, save, выбор станции и настроения.

### Trade-offs
- Strong at: самый простой и мобильный.
- Weak at: меньше «сцены», больше устройство.
