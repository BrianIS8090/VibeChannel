## Variant: Pocket TUI Radio

### Design stance
Карманный TUI для телефона: минимум панелей, но всё ещё терминальная структура.

### Key choices
- Layout: главный экран эфира + очередь + настроение + заметка.
- Typography: моноширинный интерфейс.
- Color: холодный синий терминал.
- Interaction: play/pause, scan, save, выбор станции.

### Trade-offs
- Strong at: лучшая мобильная читаемость.
- Weak at: меньше ощущения настоящего десктопного TUI.
