## Variant: Pixel Night Room

### Design stance
Пиксельная ночная комната: радио — часть уютной сцены, а не интерфейсная панель.

### Key choices
- Layout: большая пиксельная сцена + очень короткий плейлист.
- Typography: минимум текста.
- Color: тёмная комната, тёплое радио, мягкие пастельные кнопки.
- Interaction: play, next, favorite, выбор эфира.

### Trade-offs
- Strong at: самый атмосферный и не перегруженный.
- Weak at: меньше утилитарности.
