## Variant: Radio TUI Classic

### Design stance
Настоящий терминальный TUI: список станций, центральный эфир, боковой лог и заметка.

### Key choices
- Layout: трехпанельный TUI на десктопе, последовательные панели на телефоне.
- Typography: только моноширинный интерфейс.
- Color: зелёный терминал.
- Interaction: выбор станции, play/pause, scan, save, лог событий.

### Trade-offs
- Strong at: ближе всего к реальному TUI.
- Weak at: менее декоративный, больше утилита.
