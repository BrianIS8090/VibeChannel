## Variant: Pocket Night

### Design stance
Смартфон-first плеер с ночным неоном: мой рекомендуемый основной вариант.

### Key choices
- Layout: крупная карточка плеера + быстрый список станций.
- Typography: системный гротеск, плотный, современный.
- Color: графит, фиолетовый, циан, тёплый акцент.
- Interaction: play/pause, избранное, режимы, выбор/случайная станция.

### Trade-offs
- Strong at: мобильный сценарий, современный вид, понятные большие кнопки.
- Weak at: менее «ламповый», больше похож на музыкальное приложение.

### Best for
Первый MVP, если главное — телефон и ежедневное использование.
