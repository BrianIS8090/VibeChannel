## Variant: Pixel Jukebox FM

### Design stance
Пиксельный FM-джукбокс: крупный объект в центре, каналы и заметка отдельно.

### Key choices
- Layout: большая радио-башня/джукбокс + боковые лёгкие панели.
- Typography: крупные названия станций.
- Color: морской ночной фон, коралловый корпус, жёлтый дисплей.
- Interaction: play, scan, save, выбор канала и настроения.

### Trade-offs
- Strong at: самый яркий и запоминающийся.
- Weak at: чуть менее спокойный.
