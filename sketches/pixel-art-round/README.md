# VibeChannel — пиксель-арт раунд

Полная смена направления после отклонения TUI-вариантов.

Требование: не перегружать интерфейс, стиль — пиксель-арт, компоновка новая.

## Варианты

1. `../010-pixel-pocket-radio/index.html` — карманная пиксельная радио-консоль.
2. `../011-pixel-night-room/index.html` — пиксельная ночная комната с радио.
3. `../012-pixel-jukebox-fm/index.html` — пиксельный FM-джукбокс.

## Моё мнение

Лучший кандидат — `011-pixel-night-room`: он сильнее всего меняет компоновку, не перегружен и наконец ощущается как vibe radio, а не как техническая панель. `010` лучше для мобильного MVP, `012` самый яркий.
