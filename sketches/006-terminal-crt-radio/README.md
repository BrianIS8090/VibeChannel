## Variant: Terminal CRT Radio

### Design stance
Терминал внутри старого CRT-радиоприёмника: вайб ретро-железа, но без уютного дерева.

### Key choices
- Layout: большой CRT-экран + память каналов.
- Typography: моноширинный текст и технические подписи.
- Color: чёрный, кислотно-жёлто-зелёный, янтарный.
- Interaction: power/play, tune, memory, preset/mood selection.

### Trade-offs
- Strong at: самый атмосферный и узнаваемый.
- Weak at: чуть менее строгий, больше арт-объект.

### Best for
Если хочется терминальный стиль, но с характером физического устройства.
