# Мокапы VibeChannel

Открывать как обычные HTML-файлы.

## Первый раунд

1. `001-pocket-night/index.html` — гибрид Pocket Player + Night Console.
2. `002-cozy-radio/index.html` — ламповый радиоприёмник.
3. `003-night-console/index.html` — техничный ночной пульт.

## Терминальный раунд

После обратной связи сделаны три новых варианта в стиле терминала:

1. `004-terminal-command-deck/index.html` — серьёзный командный пульт.
2. `005-terminal-tui-split/index.html` — TUI/ncurses-раскладка с панелями.
3. `006-terminal-crt-radio/index.html` — CRT-радиоприёмник с терминальным экраном.

## Настоящий TUI-радио раунд

После уточнения, что нужна не просто терминальная эстетика, а структура ближе к терминальному TUI с радио-вайбом:

1. `007-radio-tui-classic/index.html` — классический TUI: станции / эфир / лог.
2. `008-radio-mixer-tui/index.html` — TUI радиомикшер: тюнер, каналы, уровни.
3. `009-pocket-tui-radio/index.html` — карманный TUI для телефона.

## Моё мнение

Сейчас лучший кандидат — `007-radio-tui-classic`: он наконец не похож на обычный веб-плеер, структура реально терминальная, а радио-вайб держится через частоты, VU-метр, станции и лог. `008` сильнее по радио-метафоре, `009` лучше по мобильной чистоте.
