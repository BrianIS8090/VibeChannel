# Мокапы VibeChannel

Открывать как обычные HTML-файлы.

## Варианты

1. `001-pocket-night/index.html` — рекомендуемый гибрид Pocket Player + Night Console.
2. `002-cozy-radio/index.html` — ламповый радиоприёмник.
3. `003-night-console/index.html` — техничный ночной пульт.

## Моё мнение

Для MVP я бы выбрал `001-pocket-night`: он лучше всего держит смартфоны, выглядит современно и не превращает проект в игрушечный ретро-приёмник.

Из `003-night-console` можно забрать терминальный шарм, но не делать весь продукт таким жёстким.
